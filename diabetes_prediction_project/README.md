#  Diabetes Prediction using Machine Learning

##  Project Description
This project predicts whether a person is diabetic or not using Machine Learning. The model is trained using Logistic Regression on the Diabetes Dataset.

##  Dataset
- Dataset Name: diabetes.csv
- Source: Kaggle
- Target Column: Outcome
  - 0 = Non-Diabetic
  - 1 = Diabetic

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Joblib
- Streamlit

##  Machine Learning Algorithm
- Logistic Regression

##  Project Files
- app.py
- diabetes_model.pkl
- diabetes_scaler.pkl
- diabetes_columns.pkl
- requirements.txt
- diabetes.csv
- README.md

streamlit run app.py
