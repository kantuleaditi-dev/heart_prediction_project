# Heart Disease Prediction

## Project Description
This project predicts whether a person has heart disease using Machine Learning and Streamlit.

## Dataset
Heart Disease Dataset

## Features
- Age
- Sex
- ChestPainType
- RestingBP
- Cholesterol
- FastingBS
- RestingECG
- MaxHR
- ExerciseAngina
- Oldpeak
- ST_Slope

## Target
HeartDisease

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Machine Learning Model
Logistic Regression

## How to Run
pip install requirements.txt
streamlit run app.py
